# JS Web3 SDK

虽然大多数现有的SDK应该开箱即用，但部署智能合约或使用独特的zkSync功能，如以其他代币支付费用，需要提供以太坊交易默认的额外字段。

为了方便访问zkSync Era的所有功能，创建了`zksync-web3`JavaScript SDK，它是基于`ethers.js`库。本节作为其参考。
